## Fix Checklist (apply to every file)

### A — Arabic Display Terms
Replace these exact strings wherever they appear as user-visible text:

  "Fact"         → "معاملات"
  "Dimension"    → "تصنيفات"
  "Summary"      → "ملخصات"
  "Bridge"       → "روابط"
  "Cardinality"  → "نوع العلاقة"
  "Proposed"     → "مقترح"
  "Confirmed"    → "مؤكد"
  "Rejected"     → "مرفوض"
  "Profile"      → "نظرة عامة"
  "Visualize"    → "الرسوم البيانية"
  "Predictions"  → "التوقعات"
  "Clusters"     → "التجميعات"
  "Data model"   → "نموذج البيانات"
  "Confidence"   → "نسبة الثقة"
  "Refresh"      → "تحديث"
  "Edit"         → "تعديل"
  "Save"         → "حفظ"
  "Cancel"       → "إلغاء"
  "Confirm"      → "تأكيد"
  "Reject"       → "رفض"
  "Reset"        → "إعادة تعيين"
  "Dismiss"      → "تجاهل"
  "high"         → "عالية"
  "medium"       → "متوسطة"
  "low"          → "منخفضة"

  Only replace when the string is rendered to the user.
  Do NOT replace inside: if conditions, API calls, type definitions, 
  className strings, or variable names.

### B — Empty States
Every empty state must have:
  - Icon (SVG)
  - Arabic title: "لا توجد بيانات بعد"
  - Arabic subtitle explaining what to do next
  - CTA button if an action is possible

### C — Error Messages
Every catch block or error display must show:
  - Plain Arabic explanation of what went wrong
  - What the user can do about it
  Never show raw error strings or stack traces.

### D — Loading States
Every button that calls an API must:
  - Be disabled during the call
  - Show a spinner + short Arabic text ("جاري الحفظ...")
  - Return to normal on success or failure

### E — RTL Layout
  - All Arabic text must be dir="rtl" or text-right
  - Directional icons (arrows, chevrons) must be mirrored for RTL:
    ChevronRight → ChevronLeft, ArrowRight → ArrowLeft
  - flex-row containers with Arabic content → flex-row-reverse

### F — Mobile (375px)
  - No horizontal scroll at 375px width
  - Minimum touch target: 44×44px for all buttons
  - Font size minimum: 14px body, 12px secondary

### G — Confidence Score Display
Wherever a confidence score (0-100) is shown:
  - Add colored gauge: ≥70 green, 40-69 yellow, <40 red
  - Add Arabic explanation below the number
  - Add what factors affected the score if available

### H — Destructive Action Protection
Before: reject relationship, delete dataset, dismiss question
  - Show Arabic confirmation: "هل أنت متأكد؟ لا يمكن التراجع عن هذا الإجراء"
  - Two buttons: "نعم، متأكد" (danger color) and "إلغاء"

### I — Success Feedback
After: save description, confirm relationship, answer question, upload file
  - Show Arabic toast: "تم الحفظ بنجاح ✓" for 3 seconds
