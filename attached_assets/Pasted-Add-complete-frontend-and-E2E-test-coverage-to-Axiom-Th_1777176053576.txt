Add complete frontend and E2E test coverage to Axiom. The backend already has
202/202 passing tests. This task adds everything that's currently
MANUAL_REVIEW_REQUIRED plus Tahleel and performance tests.

## What to install

In frontend/:
  npm install -D vitest @vitejs/plugin-react @testing-library/react \
    @testing-library/user-event @testing-library/jest-dom \
    @playwright/test msw happy-dom

In root:
  pip install locust --break-system-packages

## Step 1 — Vitest config

Create frontend/vitest.config.ts:
  - environment: happy-dom
  - setupFiles: src/tests/setup.ts
  - globals: true
  - coverage provider: v8, reporter: text + json

Create frontend/src/tests/setup.ts:
  - import @testing-library/jest-dom
  - mock next/navigation (useRouter, useSearchParams, usePathname)
  - mock next/image
  - global fetch mock that returns { ok: true, json: () => ({}) }
